### 初始化参数ServletConfig

#### DD部署文件（web.xml）

```xml
<web-app>
    <servlet>
        <servlet-name>testServlet</servlet-name>
        <servlet-class>com.bior.TestServlet</servlet-class>

        <init-param>
            <param-name>adminEmail</param-name>
            <param-value>biorson@163.com</param-value>
        </init-param>
    </servlet>

    <servlet-mapping>
        <servlet-name>testServlet</servlet-name>
        <url-pattern>test.do</url-pattern>
    </servlet-mapping>
</web-app>
```

#### Servlet代码

```java
public class TestServlet extends HttpServlet{
    
    public void doPost(HttpServeltRequest request,HttpServletResponse response){
        
        ServletConfig config = request.getServletConfig();
        String adminEmail = config.getInitParameter("adminEamil");
    }
}
```
